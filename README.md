# CLIENT

## Installation

Navigate to client folder.

Use the package manager [npm](https://www.npmjs.com/) to install dependancies.

```bash
npm install
```

## Usage

```javascript
npm start
```

Open [localhost:9000](http://localhost:9000/) to display client application

# SERVER

## Installation

Navigate to server folder.

Use the package manager [npm](https://www.npmjs.com/) to install dependancies.

```bash
npm install
```

## Usage

```javascript
npm start
```

## Reminder
